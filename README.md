# Employee Management System (EMS) – Python

## 📌 Project Overview

The **Employee Management System (EMS)** is a simple Python-based application designed to manage employee records using basic programming concepts.
This project demonstrates the use of **Python dictionaries, functions, loops, conditional statements, and menu-driven programs** to store and manage employee information.

The system allows users to add employees, view employee records, and search for employees by their ID.

---

## 🎯 Objectives

* Understand Python control structures
* Practice using dictionaries for data storage
* Implement functions for modular programming
* Build a menu-driven application

---

## ⚙️ Features

* Add a new employee
* View all employees
* Search employee by ID
* Simple command-line interface
* Data stored using Python dictionary
